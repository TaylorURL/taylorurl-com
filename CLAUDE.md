# CLAUDE.md

## Project Overview

TaylorURL is a business website for a web development company serving Baytown, Houston, and surrounding Texas cities. Built with React 19 + Vite 7 + Tailwind CSS 3 + framer-motion + lucide-react + recharts.

## Key Conventions

- **Never commit or push to git manually.** Always use `nit release` (`npm run release`) for all version bumping, changelogs, builds, and git operations.
- Path aliases: `@components`, `@views`, `@hooks`, `@constants`, `@data` (configured in vite.config.js and jsconfig.json).
